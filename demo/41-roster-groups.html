Roster groups — Crew A / Crew B with chevron toggle

Phase A1 of the schedule-page gap plan. Resource groups render a header row above their members with chevron, swatch, name and member count. Toggle the chevron to collapse / expand the crew. Resources with no explicit crew (Joe) get wrapped under a synthetic Other header so they sit visually separate from the named crews. Pass ungroupedGroupTitle: null to opt out of that bucket.

Scroll the calendar horizontally — the resource column stays glued to the left, only the day strip slides under it. There is exactly one row-head DOM tree, ever (no Pager snapshots).
