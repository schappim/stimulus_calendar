Continuous-week scroll on TimeGridWeek

Phase C1. Set continuousWeekScroll: true on a timeGridWeek view and the controller swaps in the WeekScroller instead of the standard Pager. Each visible page is a fully-rendered week TimeGrid; CSS scroll-snap locks the user to one week at a time, scroll-settle bumps options.date to the centred week. Swipe left/right (or use the trackpad) to navigate continuously.
