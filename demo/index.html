stimulus_calendar

A Stimulus.js full-sized drag & drop event calendar with resource & timeline views, plus live multi-user sync.

See PLAN.md for the per-feature checklist and README.md for the full option/method reference.

DayGrid

  • Month view Classic dayGridMonth with week numbers and toolbar.
  • Week view dayGridWeek — one row of seven day-cells.
  • Day view dayGridDay — single-day all-day grid.
  • Day popover Click +N more in a month cell to fan out the day's events.

TimeGrid

  • Week view timeGridWeek with hour-slot rendering and scrollTime.
  • 3-day view new Same component as the week view but with duration: { days: 3 }.
  • Day view timeGridDay — single day-column with time axis.

List & Resource views

  • List week listWeek — flat agenda of the week's events.
  • Resource timeline (week) Rows = resources, columns = days — horizontal Gantt-ish layout.
  • Nested resources resourceTimelineWeek with expandable parent/child resources.

Interaction

  • Drag, drop & resize Pointer-driven event move + bottom-edge resize inside timeGridWeek.
  • Swipe navigation Trackpad swipe, finger swipe, keyboard arrows; click-to-create + add-event.
  • Event popover Double-click any chip for an edit/delete popover.

Mobile

  • Mobile-optimised new Phone-shell layout, sticky top bar & bottom view switcher, FAB, safe-area insets, native date pickers in a bottom sheet, single-tap popover, fat-finger slot heights, swipe navigation.

Data & sync

  • Turbo Streams live sync Two windows synchronised in real time via the broadcast channel.
  • Overview Smoke-test page seeded with a mix of timed and all-day events.
  • Category filters eventFilter — toggle event categories on/off live.
  • Event search Filter events by title with a search input bound to eventFilter.

Forecast

  • Weather forecast new All views (month / week / day / list) with a per-day weather icon, high/low and forecast strip. Switch views in the toolbar to see how the forecast adapts.

Timezones

  • IANA timezones new Named zones like Australia/Sydney and America/New_York — DST-aware via Intl.DateTimeFormat. Live switcher plus a side-by-side world clock showing the same UTC instant in five zones.
  • Cross-timezone live sync new Two browser panes on a shared BroadcastChannel but in different zones — drag in Sydney, the same UTC instant lands at New York's local wall-clock. Wire format carries the explicit offset so receivers can re-anchor.

Tradie / field service new

A ServiceM8 / Tradify / Fergus-style dispatch board, field view and customer touchpoints layered on top of the calendar. Conventions and hook map live in TRADIE.md; shared data + helpers in tradie_data.js.

Dispatch board

  • Dispatch board The spine: resourceTimeGridDay with column-per-crew, status-coloured chips, drag-between-lanes fires a tradie:reassigned hook + toast.
  • Unscheduled job tray Sidebar of jobs waiting to be slotted; drag a card onto a lane to schedule. Pure host-app — dateFromPoint + DOM walk for the resource.
  • Status lifecycle Quoted → scheduled → in-progress → completed → invoiced. Click the chip's status pill to advance; pipeline header re-counts live.

Geography

  • Auto travel-time blocks Background events auto-inserted between back-to-back jobs at different addresses; rebuilds on every drop with km + minutes + tight-gap warning.
  • Map sidecar SVG map of today's jobs as crew-coloured pins; click pin ↔ chip selects the matching job two-way.
  • Nearby-hint on drag Floating tooltip during drag — shows distance to the closest existing job in whichever lane you're hovering over.

Tradie reality

  • Arrival windows Customer "between 9–11am" rendered as a hatched band over the booked chip; chip can move inside the band without losing the promise.
  • Multi-day site work + service calls Builder on a 3-day site rendered as a striped banner across the week timeline alongside same-day service calls in the dispatcher view.
  • On-call / after-hours shifts On-call windows render as red-hatched bands; jobs that land inside inherit an "after-hours rate" tag automatically.
  • Recurring maintenance Series expanded by the host app; click a recurring chip for "skip this occurrence", "reschedule this one", or "cancel series".
  • Job dependencies Fit-off depends on rough-in; dropping the dependent earlier than its dependency reverts and warns. Same guard runs both directions.

Field view

  • Single-tradie agenda (mobile) Phone-shell layout, day strip, big cards with tap-to-call, tap-to-navigate, gate code / notes inline. Swap which crew you are with the picker.
  • Clock on/off → actuals Tap to clock on a job; running timer pulses on the chip; clock off writes a green "actual" band so the office sees variance at a glance.
  • Printable run sheet Paper-friendly day list per crew with travel between stops, gate codes, customer phones. @media print strips the controls.

Capacity & customer hooks

  • Capacity heatmap Crew × day grid coloured by % of working hours booked. Click any cell to drill in to that crew's day in the dispatch view.
  • Find me a slot Given duration + suburb + role, scans every crew's gaps for the next N days; surfaces top 3 by earliest-and-nearest score and pins them on the grid.
  • SMS reminders that auto-shift Each job has a child 24h-prior reminder; eventDrop on the job re-computes and updateEvents the reminder.
  • Customer self-serve confirm / reschedule Right pane is the customer-facing booking page; Confirm or Reschedule round-trips into the dispatcher view alongside.
  • Offline edit queue Toggle offline → edits queue locally with "pending sync" badges; toggle online → progress strip drains the queue.

Schedule-page gap plan new

Phases A–F of plans/schedule-page-feature-gaps.md — additive primitives to lift the existing ResourceTimeline into a full Roster lens (crews, empty-cell add, NOW vertical line, drag-to-reassign) and round out the Calendar lens (continuous-week scroll, density dots, stripe chips, confirmation states, scheduling-X mode).

  • A1 · Roster resource groups Crew header rows with chevron, swatch, member count + JS-driven setGroupExpanded. Ungrouped resources render flat.
  • A2 · Sticky day-axis header Day-axis stays parked while crews scroll past; row-head column glues left during horizontal scroll. Bounded-height container.
  • A3 · Empty-cell quick-add Hover a strip cell for the ＋; tap fires calendar:cellClick with date + resource + group so the host can open the new-appointment sheet pre-filled.
  • A4 · Today band + live NOW line Today's column tinted across every crew row; vertical NOW rule ticks every second via state.now without re-rendering.
  • A5/A6 · Bar resize + drag-to-reassign Drag a bar horizontally to shift; drag its right edge to grow; drop on a different crew's row to reassign — eventDrop gains oldResource / newResource / newResourceIds and round-trips through updateEvent.
  • A7 · Narrow auto-class Below eventNarrowThreshold a bar earns .ec-event-narrow; default CSS hides secondary text. Live slider re-flows bars across the threshold via ResizeObserver.
  • B · Hour-column, today-circle, row pinch slotMode: 'hours' swaps the day-strip for per-day hour columns (with lunch shading); setRowHeight cycles compact / comfy; dayHeaderTodayStyle: 'circle' wraps the day number in an accent disc.
  • B2 · 28-day compressed Gantt resourceTimelineMonth4w renders 4 weeks of daily columns with stronger week-boundary lines; narrow auto-class keeps tight bars legible.
  • C1 · Continuous-week scroll on TimeGrid Opt-in continuousWeekScroll: true swaps the Pager for a horizontally-snapping multi-week stack; mirrors continuousMonthScroll from DayGrid.
  • C2/C3/C5/C6 · Density dots, stripe chips, states dayHeaderDensity dots on the weekday column heads; dayCellEventStyle: 'stripe' Month-view chips; confirmation-state + conflict + recurring auto-classes from extendedProps.
  • D · Scheduling-X mode + suggested slot setMode writes data-calendar-mode on the host; cellAffordanceWhen lights up every cell in the active mode; setSuggestedSlot paints a pulsing slot the user can tap to commit.
