stimulus_calendar

A Stimulus.js full-sized drag & drop event calendar with resource & timeline views, plus live multi-user sync.

See PLAN.md for the per-feature checklist and README.md for the full option/method reference.

DayGrid

  • Month view Classic dayGridMonth with week numbers and toolbar.
  • Week view dayGridWeek — one row of seven day-cells.
  • Day view dayGridDay — single-day all-day grid.
  • Day popover Click +N more in a month cell to fan out the day's events.

TimeGrid

  • Week view timeGridWeek with hour-slot rendering and scrollTime.
  • 3-day view new Same component as the week view but with duration: { days: 3 }.
  • Day view timeGridDay — single day-column with time axis.

List & Resource views

  • List week listWeek — flat agenda of the week's events.
  • Resource timeline (week) Rows = resources, columns = days — horizontal Gantt-ish layout.
  • Nested resources resourceTimelineWeek with expandable parent/child resources.

Interaction

  • Drag, drop & resize Pointer-driven event move + bottom-edge resize inside timeGridWeek.
  • Swipe navigation Trackpad swipe, finger swipe, keyboard arrows; click-to-create + add-event.
  • Event popover Double-click any chip for an edit/delete popover.

Data & sync

  • Turbo Streams live sync Two windows synchronised in real time via the broadcast channel.
  • Overview Smoke-test page seeded with a mix of timed and all-day events.
  • Category filters eventFilter — toggle event categories on/off live.
  • Event search Filter events by title with a search input bound to eventFilter.

Forecast

  • Weather forecast new All views (month / week / day / list) with a per-day weather icon, high/low and forecast strip. Switch views in the toolbar to see how the forecast adapts.

Timezones

  • IANA timezones new Named zones like Australia/Sydney and America/New_York — DST-aware via Intl.DateTimeFormat. Live switcher plus a side-by-side world clock showing the same UTC instant in five zones.
  • Cross-timezone live sync new Two browser panes on a shared BroadcastChannel but in different zones — drag in Sydney, the same UTC instant lands at New York's local wall-clock. Wire format carries the explicit offset so receivers can re-anchor.
